# AIとはなにか
**AIは一言で言うと、人間のやるような動作や振る舞いをコンピューターで再現するシステムです。**  
なので、ただ天気の情報をサイトから取得してきて値表示させる、それはAIではありません。  
しかし、値を取得して湿度などの相関関係などを出力したりする、それがAIです。

AIはいくつかの要素で、どこまで信憑性があるかや予測を行います。

# 線形回帰　非線形回帰とは
その時のデータが本当に相関関係があるのかなどを確認しないと、誤った学習をさせてしまいます。  
データ内にハズレ値やノイズがある場合、その時に分析をするものがあります。  
それが**線形回帰**と**非線形回帰**です（他にもあります）。

**線形回帰とは**  
![線形回帰グラフ](image.png)  
データの中にある関係値を直線します。

**非線形回帰とは**  
![非線形回帰](image-1.png)  
データの中にある関係を曲線にし表します。

## 線形回帰とは
線形回帰とは、直線的な関係の数値のときに使います。  
比較的直線の関係になるようなものに使われます。  
例: **日照時間と植物の発育**など。

目的変数と説明変数がどれだけ影響を与えるかを予測するものです。

## 非線形回帰
非線形回帰は、直線では説明できない曲がったデータのときに使います。

### どう使い分けるか
**非線形曲線**  
- 塩の溶解度  
- バッテリー残量と電圧  

**線形曲線**  
- 睡眠時間と集中力テストの点数（6〜8時間）  
- 勉強時間とテストの点数  

## どのような数式で表すのか見ていきましょう
**線形回帰**  
```
y = ax + b
```
 
- y は目的変数（予測したい数字）  
- x は説明変数（入力した値）  
- a は傾き（変化の割合）  
- b は切片（直線がy軸と交わる点）
です

**非線形回帰では複数のモデルが存在します。代表的なもの**  
- 指数関数モデル: `y = a × e^(bx)`  
- 多項式モデル: `y = ax² + bx + c`  
- ロジスティックモデル: `y = L / (1 + e^(-k(x - x₀)))`  

ここで、a, b, c, k, L, x₀ は値です.

# シグモイド関数やニューラルネットワークについて
シグモイド関数は、**非線形回帰の一つのもの**で、ニューラルネットワークでよくでてくる考え方です
シグモイド関数は、どんな数を入れても必ず0から1の間の数に変換するものです
例として、テストの点数が100点満点だった時に、それを0から1の範囲（パーセンテージ）に変換するのに似ています。
つまりどのくらい〇〇であるかなどが判定できます

### シグモイド関数の数式
![シグモイド関数図](image-2.png)
`σ(x) = 1 / (1 + e^(-x))`
この関数の特徴：
- **値が小さい時**: 出力は0に近づく
- **値が0**: 出力は0.5
- **値が大きい時**: 出力は1に近づく
## ニューラルネットワークとは？
ニューラルネットワークは、人間の脳の仕組みを真似して作られたコンピュータの仕組み。  
人間の脳にある細胞が、情報を伝える様子を真似して作られました。
![ニューラルネットワークの図](image-4.png)

### 働く流れ
1. **情報を受け取る**: 複数の情報（数字）が入ってくる
2. **情報を合計する**: 受け取った情報を全部足し算する
3. **判断する**: シグモイド関数を使って「YES」か「NO」かを決める
4. **結果を伝える**: 次の神経細胞に結果を送る

![ニューラルネットワークのシナプスの図](image-3.png)

## 実際の応用例を見てみよう
### 身近なAIの使用例
- **スマートフォンの顔認証**: 顔の特徴を数値化して本人かどうかを判断
- **音声アシスタント**: 音声を文字に変換し、質問内容を理解
- **推薦システム**: 過去の購入履歴から好みの商品を予測

### 線形回帰と非線形回帰の使い分けの具体例
**線形回帰が適している場面**
- 広告費と売上の関係（広告費が増えると売上も比例して増加）
- 気温と電力消費量の関係（暖房・冷房使用量）

**非線形回帰が適している場面**
- 薬の投与量と効果の関係（一定量を超えると効果が頭打ち）
- 人口増加の予測（指数関数的な成長）

## AIが学習する仕組み
### データから学習するプロセス
1. **大量のデータを準備**: 正解がわかっているデータを用意
2. **予測を実行**: AIが現在の知識で予測を行う
3. **間違いを確認**: 予測結果と正解を比較
4. **調整を行う**: 間違いを減らすように重みやバイアスを調整
5. **繰り返し学習**: 2〜4を何千回も繰り返す

### なぜAIは間違いから学習できるのか
人間が自転車に乗る練習と似ています：
- 最初は倒れてしまう（間違い）
- 倒れる原因を分析する（エラーの確認）
- バランスの取り方を調整する（重みの調整）
- 何度も練習して上達する（反復学習）

## AIの限界と注意点
### AIができないこと
- **感情の理解**: 人間の複雑な感情を完全に理解することは困難
- **創造性**: 全く新しいアイデアを生み出すのは苦手
- **常識的判断**: 人間にとって当たり前のことが理解できない場合がある

### AIを使う時の注意点
- **データの質**: 間違ったデータで学習すると間違った結果を出す
- **偏見の問題**: 偏ったデータで学習すると偏った判断をする
- **説明の難しさ**: なぜその結論に至ったかを説明するのが困難

## まとめ
AIは線形回帰や非線形回帰などの数学的手法を組み合わせて、人間のような判断を行います。
シグモイド関数やニューラルネットワークは、この判断を可能にする重要な仕組みです。
しかし、AIにも限界があることを理解して、適切に活用することが大切です。
