# スマホ改造ガイド

このガイドでは、**AU版 Sony Xperia XZ2 Premium (SOV38)** を例に、Androidスマホの改造方法をステップバイステップで解説します。

## このガイドでできるようになること

- **ブートローダーアンロック** — カスタムROMやroot化の前提条件
- **root化 (Magisk)** — スマホの管理者権限を取得し、自由にカスタマイズ
- **カスタムリカバリ (TWRP)** — バックアップや高度な操作が可能に

## 対象端末

このガイドは以下の端末で実際に行った作業をベースにしています。

| 項目 | 値 |
|------|-----|
| 機種名 | Sony Xperia XZ2 Premium |
| 型番 | SOV38 (AU版) |
| コードネーム | aurora_kddi |
| SoC | Qualcomm Snapdragon 845 |
| Android | 10 |

:::tip 他の機種でも参考になる
手順の細部は機種によって異なりますが、**考え方や流れ**はほとんどのAndroid端末で共通です。
:::

## 改造の全体像

スマホ改造は、以下のステップで進めます：

```
Step 1: 準備（ツール導入・バックアップ）
    ↓
Step 2: ブートローダーアンロック（BLU）
    ↓
Step 3: カスタムリカバリ（TWRP）導入
    ↓
Step 4: root化（Magisk）
    ↓
完了！ 自由にカスタマイズ可能に
```

## 改造前に知っておくべきこと

### メリット

- システムレベルのカスタマイズが自由に
- 広告ブロック、テーマ変更、CPU制御など
- 不要なプリインアプリの削除
- カスタムROMで最新Androidを使える可能性

### デメリット・リスク

:::danger 改造のリスク
以下のリスクを理解した上で進めてください：

- **メーカー保証が無効** になる
- **文鎮化（起動不能）** のリスクがある
- Sony端末では **カメラのDRM機能が失われる**（画質低下）
- **Widevine L1 → L3** に低下（Netflix等のHD再生不可）
- **おサイフケータイ / FeliCa** が使えなくなる場合がある
- 操作を誤ると **復旧不可能** になることもある
:::

### 用語集

初めての人向けに、このガイドで使う用語をまとめました：

| 用語 | 意味 |
|------|------|
| **ブートローダー (BL)** | スマホの電源を入れたとき最初に動くプログラム。OSを起動する役割 |
| **BLU (BL Unlock)** | ブートローダーのロックを解除すること。改造の第一歩 |
| **root化** | スマホの管理者権限（root）を取得すること。PCでいう「Administrator」 |
| **Magisk** | systemを書き換えずにrootを取得するツール。SafetyNetも回避可能 |
| **TWRP** | カスタムリカバリ。バックアップやZIPの書き込みができる |
| **文鎮化（ブリック）** | スマホが起動しなくなること。「高級な文鎮」になる |
| **TAパーティション** | Sony端末固有の領域。DRMキーやシリアル番号を保存 |
| **ファームウェア** | スマホのOS・ドライバなどソフトウェア全体 |
| **adb** | PCからスマホを操作するツール (Android Debug Bridge) |
| **fastboot** | ブートローダーモードでPCからスマホを操作するツール |

## 必要な環境

- **PC**: Windows / macOS / Linux（このガイドではmacOSを使用）
- **USBケーブル**: スマホとPCを接続できるもの
- **安定したインターネット接続**
- **時間**: 初回は2〜3時間見ておくと安心
